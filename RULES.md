# Aiaこんぶ ツール部 開発ルール
最終更新：2026年4月

━━━━━━━━━━━━━━━━━━━━━━━━
1. ファイル命名規則
━━━━━━━━━━━━━━━━━━━━━━━━

形式：`NNN-tool-name.html`

- NNN：3桁ゼロ埋め番号（例：001, 007）
- tool-name：英語・ハイフン区切り・小文字
- 拡張子：.html（React使用時は.jsx）

例）
005-unit-converter.html
006-area-calculator.html
007-observation-notebook.html

━━━━━━━━━━━━━━━━━━━━━━━━
2. バージョン規則
━━━━━━━━━━━━━━━━━━━━━━━━

- 初回完成　　：v1.0
- 軽微な修正　：v1.1 / v1.2 …
- 機能追加　　：v2.0

バージョンはツール内ヘッダーに必ず表示すること。

━━━━━━━━━━━━━━━━━━━━━━━━
3. ツール内に必ず含める要素
━━━━━━━━━━━━━━━━━━━━━━━━

- バッジ　　：No.NNN
- タイトル　：ツール名
- バージョン：v1.0 など
- 説明文　　：一文で何ができるか

━━━━━━━━━━━━━━━━━━━━━━━━
4. index.html 更新ルール
━━━━━━━━━━━━━━━━━━━━━━━━

- ツール完成ごとにindex.htmlへの追記が必要
- スマホ作業中はPCのタイミングにまとめて実施してOK
- ツール数カウント（`X tools`）も忘れず更新する

━━━━━━━━━━━━━━━━━━━━━━━━
5. UIパターン：オプションパネル
━━━━━━━━━━━━━━━━━━━━━━━━

設定・補助項目は「⚙ オプション」ボタンで折りたたむ。

【実装ルール】
- ボタンラベル：「⚙ オプション」固定
- ボタンクラス：toggle-btn（既存スタイル流用）
- 展開時：activeクラスを付与
- パネル：display:none → display:flex で切り替え
- パネル内はflex・wrap・gap:6pxで横並び

【使いどころ】
- 小数点桁数の選択
- 出力フォーマットの切り替え
- その他、メインの操作に影響しない補助設定

【参考実装】No.002 座標計算ツール v1.1
